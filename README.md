# services-website

Scaffolded Vite + React (TypeScript) marketing site. Layout imitates intelivita.com with placeholder content.

Quick start
1. npm install
2. npm run dev

Build
- npm run build
- npm run preview

What I implemented
- Vite + React (TypeScript) scaffold
- Tailwind CSS configuration
- Header, Footer, Hero, Services grid, Clients logos, Case studies page, Contact form (client-side)
- Basic SEO meta tags, sitemap.xml, robots.txt

Assets & final content you should provide
- High-resolution logo (replace `public/images/logo.svg`)
- Hero image(s) and case study images (replace `public/images/hero-placeholder.png`)
- Client logos (replace `public/images/client-*.svg`)
- Final copy for hero, services, case studies, footer, and meta descriptions
- Contact form backend endpoint (currently a client-side stub)

Deployment
- This app is a static single-page app and can be deployed to Netlify, Vercel (static), or any static host that serves `index.html`.
